# react-dev-test

	git clone git@github.com:passion0805/react-dev-test.git

	react-dev-test folder:
		1. npm install
		2. npm run serve
		3. `localhost:30080` in your browser
		
		
Tech stacks:
	``react (v16)``
	``axios``
	``bootstrap (v4)``
	``node-sass``
	``react-redux``
	``redux (v4)``
	``react-custom-scrollbars``
	``reselect``
