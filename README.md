# react-dev-test


Run with 30080 port:

	git clone this repo

	inside react-dev-test folder:
		1. npm install
		2. npm run serve
		3. input `localhost:30080` in your browser
		
		
Tech stacks:
	``react (v16)``
	``axios``
	``bootstrap (v4)``
	``node-sass``
	``react-redux``
	``redux (v4)``
	``react-custom-scrollbars``
	``reselect``
